# AIToken Cookbook

七牛云 [AIToken](https://developer.qiniu.com/aitokenapi) 平台 API 使用示例与指南，帮助开发者快速上手各类 AI 模型的调用。

运行这些示例前，你需要一个七牛 AIToken 账号和 API Key（[点此获取](https://developer.qiniu.com/aitokenapi/12884/how-to-get-api-key)）。设置环境变量 `QINIU_API_KEY` 为你的 API Key：

```bash
export QINIU_API_KEY="your-api-key"
```

大多数代码示例是用Python编写的，尽管这些概念可以应用于任何语言。

更多信息请参阅 [AIToken 文档](https://developer.qiniu.com/aitokenapi) 和 [API 参考](https://apidocs.qnaigc.com)。
